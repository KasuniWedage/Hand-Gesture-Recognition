# Hand Gesture Image Classification System

ICT 3212 – Introduction to Intelligent Systems
Rajarata University of Sri Lanka | Department of Computing

---

## Project Overview
This project develops an intelligent image classification system capable of
automatically recognizing 20 different hand gestures from images using a
Convolutional Neural Network (CNN).

---

## Dataset
- Source: Hand Gesture Recognition Dataset (Kaggle)
- Total Images: 24,000
- Classes: 20 (labeled 0–19)
- Training: 900 images per class (18,000 total)
- Testing: 300 images per class (6,000 total)

---

## Model
- Architecture: Convolutional Neural Network (CNN)
- Input Size: 64 × 64 × 3
- Optimizer: Adam
- Loss Function: Categorical Cross-Entropy
- Epochs: 20
- Batch Size: 64

---

## Results (Implementation 1)
- Test Accuracy: 99.98%
- Misclassifications: 1 out of 6,000 test images
